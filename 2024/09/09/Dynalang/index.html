Dynalang

World Model Learning

Dynalang predicts the future in a multimodal world model, which grounds diverse types of language to visual experience of embodied agent. Viewed as an observation signal, natural language can be use to

  1. specific task (what to do)
  2. provide domain prior knowledge (how the world behaves)
  3. update belief (what we know about the state of the world)
  4. Q & A on embodied question
  5. … …

The multimodal RSSM world model in Dynalang consists of the following components

Similar to Dreamer v3, the stochastic latent state is categorically distributed. Rewards are predicted via a softmax classifier with exponentially spaced bins that regresses the twohot encoding of the real-valued rewards

The world model is trained to optimize the overall loss with respect to all its parameters

The world model can be pretrained on offline text-only dataset by zeroing out image and action input and discarding the image, reward and continue prediction loss items, which can be finetuned further
