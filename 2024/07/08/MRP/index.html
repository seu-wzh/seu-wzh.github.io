Markov Reward Process

马尔可夫奖励过程（MRP）

基础定义

一个 MRP 模型 M\mathscr{M}M 可以形式化定义为元组 ⟨S, P, R, γ⟩\langle \mathcal{S},\ \mathcal{P},\ \mathcal{R},\ \gamma \rangle⟨S, P, R, γ⟩：

  1. 有限状态集合 S={s1, s2, ⋯ , sn}\mathcal{S} = \{ s_{1},\ s_{2},\ \cdots,\ s_{n} \}S={s1​, s2​, ⋯, sn​}
  2. 状态转移概率矩阵 P={pij=p(St+1=si∣St=sj)}n×n\mathcal{P} = \{ p_{ij} = p(S_{t + 1} = s_{i} \mid S_{t} = s_{j}) \}_{n \times n}P={pij​=p(St+1​=si​∣St​=sj​)}n×n​
  3. 奖励函数 R:S↦R=E(Rt+1∣St=s)\mathcal{R} : \mathcal{S} \mapsto \mathbb{R} = \mathcal{E}(R_{t + 1} \mid S_{t} = s)R:S↦R=E(Rt+1​∣St​=s)
  4. 衰减因子 γ∈[0, 1)\gamma \in [0,\ 1)γ∈[0, 1)

其中状态集合 S\mathcal{S}S 和 状态转移概率矩阵 P\mathcal{P}P 的定义和普通的马尔可夫过程一致。奖励函数 R\mathcal{R}R 代表某时刻 ttt 处于状态 sss 时获得的收益的期望（下一时刻收取）。考虑在某一时刻 ttt 处于状态 sss 时未来一段时间 T\mathrm{T}T 的回报：

Gt(T, 1)=Rt+1+Rt+2+⋯+Rt+TG_{t}(\mathrm{T},\ 1) = R_{t + 1} + R_{t + 2} + \cdots + R_{t + \mathrm{T}} Gt​(T, 1)=Rt+1​+Rt+2​+⋯+Rt+T​

考虑近期收益和远期收益在实际情况中并不等价，因此使用衰减因子 γ\gammaγ 对未来不同时间的收益进行加权：

Gt(T, γ)=Rt+1+γRt+2+⋯+γTRt+TG_{t}(\mathrm{T},\ \gamma) = R_{t + 1} + \gamma R_{t + 2} + \cdots + \gamma^{\mathrm{T}} R_{t + \mathrm{T}} Gt​(T, γ)=Rt+1​+γRt+2​+⋯+γTRt+T​
