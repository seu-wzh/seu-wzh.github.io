Partially Observable Markov Decision Process

部分可观察马尔可夫决策过程（POMDP）

基础定义

一个 POMDP 模型 M\mathscr{M}M 可以形式化定义为元组 ⟨S, A, O, P, Q, R, γ, b0⟩\langle \mathcal{S},\ \mathcal{A},\ \mathcal{O},\ \mathcal{P},\ \mathcal{Q},\ \mathcal{R},\ \gamma,\ b_{0} \rangle⟨S, A, O, P, Q, R, γ, b0​⟩：

  1. 有限状态集合 S={s1, s2, ⋯ , sn}\mathcal{S} = \{ s_{1},\ s_{2},\ \cdots,\ s_{n} \}S={s1​, s2​, ⋯, sn​}
  2. 有限动作集合 A={a1, a2, ⋯ , am}\mathcal{A} = \{ a_{1},\ a_{2},\ \cdots,\ a_{m} \}A={a1​, a2​, ⋯, am​}
  3. 有限观测集合 Ω={o1, o2, ⋯ , ol}\Omega = \{ o_{1},\ o_{2},\ \cdots,\ o_{l} \}Ω={o1​, o2​, ⋯, ol​}
  4. 状态转移概率矩阵 P={pij(a)=p(St+1=si∣St=sj, At=a)}m×n×n\mathcal{P} = \{ p_{ij}^{(a)} = p(S_{t + 1} = s_{i} \mid S_{t} = s_{j},\ A_{t} = a) \}_{m \times n \times n}P={pij(a)​=p(St+1​=si​∣St​=sj​, At​=a)}m×n×n​
  5. 观测状态激发概率矩阵 O={pij=p(Ot=oi∣St=sj)}l×n\mathcal{O} = \{ p_{ij} = p(O_{t} = o_{i} \mid S_{t} = s_{j}) \}_{l \times n}O={pij​=p(Ot​=oi​∣St​=sj​)}l×n​
  6. 奖励函数 R:S×A↦R=E(Rt+1∣St=s, At=a)\mathcal{R} : \mathcal{S} \times \mathcal{A} \mapsto \mathbb{R} = \mathcal{E}(R_{t + 1} \mid S_{t} = s,\ A_{t} = a)R:S×A↦R=E(Rt+1​∣St​=s, At​=a)

信息状态通过每次采取的动作 AtA_{t}At​ 和新观测到的 Ot+1O_{t + 1}Ot+1​ 进行更新，并且要求这种更新具有马尔可夫性，即：
