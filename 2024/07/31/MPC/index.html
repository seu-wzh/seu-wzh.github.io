Model Predictive Control（MPC）

与 model-free 方法相比，MPC 方法并不构建一个显式的策略，而是根据已知的或习得的环境模型从当前状态 sts_{t}st​ 开始，对多个候选动作序列进行推演，并选取回报最大的序列的第一个动作来执行：

ΠθMPC(st)=max⁡at:t+H∼pθ(⋅)Est+1∼p(⋅∣st, at)Est+2∼p(⋅∣st+1, at+1)⋯Est+H∼p(⋅∣sH−1, aH−1)[∑τ=0HγτR(st+τ, at+τ)]\Pi_{\theta}^{\mathrm{MPC}}(s_{t}) = \max_{a_{t : t + H} \sim p_{\theta}(\cdot)} \mathcal{E}_{s_{t + 1} \sim p(\cdot \mid s_{t},\ a_{t})} \mathcal{E}_{s_{t + 2} \sim p(\cdot \mid s_{t + 1},\ a_{t + 1})} \cdots \mathcal{E}_{s_{t + H} \sim p(\cdot \mid s_{H - 1},\ a_{H - 1})} \left[ \sum_{\tau = 0}^{H} \gamma^{\tau} \mathcal{R}(s_{t + \tau},\ a_{t + \tau}) \right] ΠθMPC​(st​)=at:t+H​∼pθ​(⋅)max​Est+1​∼p(⋅∣st​, at​)​Est+2​∼p(⋅∣st+1​, at+1​)​⋯Est+H​∼p(⋅∣sH−1​, aH−1​)​[τ=0∑H​γτR(st+τ​, at+τ​)]

其中 HHH 为前向推演的长度，pθ(⋅)p_{\theta}(\cdot)pθ​(⋅) 为生成候选动作序列的分布。

随机打靶方法

随机打靶方法在生成候选动作序列时直接在动作空间中进行随机采样出 NNN 条序列，通过比较选出最优动作。

Cross Entropy Method（CEM）

与随机打靶不同，CEM 维护一个带参数的分布，例如 {N(μτ, Στ)}τ=0H\Big\{ \mathcal{N}(\mu_{\tau},\ \Sigma_{\tau}) \Big\}_{\tau = 0}^{H}{N(μτ​, Στ​)}τ=0H​，并迭代提升分布采样质量：

  1. 通过分布 {N(μτ, Στ)}τ=0H\Big\{ \mathcal{N}(\mu_{\tau},\ \Sigma_{\tau}) \Big\}_{\tau = 0}^{H}{N(μτ​, Στ​)}τ=0H​， 采样出 NNN 条长度为 HHH 的动作序列 A1, A2, ⋯ , AN\boldsymbol{A}_{1},\ \boldsymbol{A}_{2},\ \cdots,\ \boldsymbol{A}_{N}A1​, A2​, ⋯, AN​
  2. 通过环境模型 P\mathcal{P}P 和 R\mathcal{R}R 对以上动作序列进行 rollout 后筛选出 NeN_{e}Ne​ 条最优的动作序列 A1, A2, ⋯ , ANe\boldsymbol{A}_{1},\ \boldsymbol{A}_{2},\ \cdots,\ \boldsymbol{A}_{N_{e}}A1​, A2​, ⋯, ANe​​
  3. 利用筛选出的最优动作序列对分布参数进行更新：

μτ←1Ne∑k=1Neat+τΣτ←1Ne∑k=1Ne(at+τ−μτ)(at+τ−μτ)⊤\mu_{\tau} \leftarrow \frac{1}{N_{e}} \sum_{k = 1}^{N_{e}} a_{t + \tau} \qquad \Sigma_{\tau} \leftarrow \frac{1}{N_{e}} \sum_{k = 1}^{N_{e}} (a_{t + \tau} - \mu_{\tau}) (a_{t + \tau} - \mu_{\tau})^{\top} μτ​←Ne​1​k=1∑Ne​​at+τ​Στ​←Ne​1​k=1∑Ne​​(at+τ​−μτ​)(at+τ​−μτ​)⊤
