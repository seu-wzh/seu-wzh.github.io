Monte-Carlo and Temporal Difference

蒙特卡洛与时序差分

动态规划算法建立在已知或习得的环境模型的基础上，而对于大部分强化学习环境来说其环境模型未知并且难以学习，也就无法通过动态规划求解，此时智能体只能通过环境交互数据来直接或间接学习到最优策略。

蒙特卡洛方法（Monte-Carlo）

蒙特卡洛算法是一大类随机算法的总称，在后续的强化学习算法中被广泛应用，例如：

近似期望

为了估计随机变量函数 f(x):Ω↦Rf(\boldsymbol{x}) : \Omega \mapsto \mathbb{R}f(x):Ω↦R 关于服从分布 p(x)p(\boldsymbol{x})p(x) 的随机变量 x\boldsymbol{x}x 的数学期望：

Ex∼p(x)[f(x)]=∫Ωf(x)p(x)dx\mathcal{E}_{\boldsymbol{x} \sim p(\boldsymbol{x})} \Big[ f(\boldsymbol{x}) \Big] = \int_{\Omega} f(\boldsymbol{x}) p(\boldsymbol{x}) d\boldsymbol{x} Ex∼p(x)​[f(x)]=∫Ω​f(x)p(x)dx

可以通过分布 p(x)p(\boldsymbol{x})p(x) 进行独立非均匀采样得到 Xn={x1, x2, ⋯ , xn}\mathcal{X}_{n} = \{ \boldsymbol{x}_{1},\ \boldsymbol{x}_{2},\ \cdots,\ \boldsymbol{x}_{n} \}Xn​={x1​, x2​, ⋯, xn​}，通过样本函数值的均值来近似期望：

qn=1n∑i=1nf(xi)≈Ex∼p(x)[f(x)]q_{n} = \frac{1}{n} \sum_{i = 1}^{n} f(\boldsymbol{x}_{i}) \approx \mathcal{E}_{\boldsymbol{x} \sim p(\boldsymbol{x})} \Big[ f(\boldsymbol{x}) \Big] qn​=n1​i=1∑n​f(xi​)≈Ex∼p(x)​[f(x)]

在实际中通常使用增量更新的方式进行计算：

进一步使用学习率 αn\alpha_{n}αn​ 来替换式中的 1n\dfrac{1}{n}n1​ 得到 qn=qn−1+αn(f(xn)−qn−1)q_{n} = q_{n - 1} + \alpha_{n} \Big( f(\boldsymbol{x}_{n}) - q_{n - 1} \Big)qn​=qn−1​+αn​(f(xn​)−qn−1​)，其中 α\alphaα 需要满足以下性质：

随机梯度

当 x\boldsymbol{x}x 服从分布 p(x)p(\boldsymbol{x})p(x) 时，为了求解以下形式的优化问题（例如神经网络）：

min⁡θEx∼p(x)[ℓ(x; θ)]\min_{\theta} \mathcal{E}_{\boldsymbol{x} \sim p(\boldsymbol{x})} \Big[ \ell(\boldsymbol{x};\ \theta) \Big] θmin​Ex∼p(x)​[ℓ(x; θ)]

在损失函数 ℓ\ellℓ 可微时，可以利用梯度下降对模型参数 θ\thetaθ 进行更新，其中对参数 θ\thetaθ 的梯度为：

g=∇θEx∼p(x)[ℓ(x; θ)]=Ex∼p(x)[∇θℓ(x; θ)]\boldsymbol{g} = \nabla_{\theta} \mathcal{E}_{\boldsymbol{x} \sim p(\boldsymbol{x})} \Big[ \ell(\boldsymbol{x};\ \theta) \Big] = \mathcal{E}_{\boldsymbol{x} \sim p(\boldsymbol{x})} \Big[ \nabla_{\theta} \ell(\boldsymbol{x};\ \theta) \Big] g=∇θ​Ex∼p(x)​[ℓ(x; θ)]=Ex∼p(x)​[∇θ​ℓ(x; θ)]

利用蒙特卡洛近似期望的方法，通过一个批次的样本集合 Xb={x1, x2, ⋯ , xb}\mathcal{X}_{b} = \{ \boldsymbol{x}_{1},\ \boldsymbol{x}_{2},\ \cdots,\ \boldsymbol{x}_{b} \}Xb​={x1​, x2​, ⋯, xb​} 即可对梯度进行估计

g^=1b∑i=1b∇θℓ(xi; θ)\hat{\boldsymbol{g}} = \frac{1}{b} \sum_{i = 1}^{b} \nabla_{\theta} \ell(\boldsymbol{x}_{i};\ \theta) g^​=b1​i=1∑b​∇θ​ℓ(xi​; θ)

蒙特卡洛策略评估

为了评估一个策略 π\piπ 的状态价值函数 vπ(s)=E(Gt∣St=s)v_{\pi}(s) = \mathcal{E}(G_{t} \mid S_{t} = s)vπ​(s)=E(Gt​∣St​=s)，可以通过 π\piπ 采样若干条序列：

s0(i)⟶a0(i)r1(i), s1(i)⟶a1(i)r2(i), s2(i)⟶a2(i)⋯⟶aT−1(i)rT(i), sT(i)s_{0}^{(i)} \overset{a_{0}^{(i)}}{\longrightarrow} r_{1}^{(i)},\ s_{1}^{(i)} \overset{a_{1}^{(i)}}{\longrightarrow} r_{2}^{(i)},\ s_{2}^{(i)} \overset{a_{2}^{(i)}}{\longrightarrow} \cdots \overset{a_{\mathrm{T} - 1}^{(i)}}{\longrightarrow} r_{\mathrm{T}}^{(i)},\ s_{\mathrm{T}}^{(i)} s0(i)​⟶a0(i)​​r1(i)​, s1(i)​⟶a1(i)​​r2(i)​, s2(i)​⟶a2(i)​​⋯⟶aT−1(i)​​rT(i)​, sT(i)​

通过采样的序列中以状态 sts_{t}st​ 为初始状态的局部子序列获得的局部回报 gtg_{t}gt​ 来估计回报的期望：

n(st)←n(st)+1v(st)←v(st)+1n(st)(gt−v(st))n(s_{t}) \leftarrow n(s_{t}) + 1 \qquad v(s_{t}) \leftarrow v(s_{t}) + \frac{1}{n(s_{t})} \Big( g_{t} - v(s_{t}) \Big) n(st​)←n(st​)+1v(st​)←v(st​)+n(st​)1​(gt​−v(st​))

同样地可以使用超参数 α\alphaα 来代替 1n\dfrac{1}{n}n1​，α\alphaα 也可以取做常数，此时更新方式不再严格遵循蒙特卡洛估计方法：

v(st)←v(st)+α(gt−v(st))v(s_{t}) \leftarrow v(s_{t}) + \alpha \Big( g_{t} - v(s_{t}) \Big) v(st​)←v(st​)+α(gt​−v(st​))

时序差分方法（Temporal Difference）

单步时序差分

在使用蒙特卡洛方法估计某个策略 π\piπ 的状态价值函数时需要等待序列采样完毕才能够进行计算，而通过时序差分方法可以在每次状态转移时通过转移四元组 (st, at, rt+1, st+1)(s_{t},\ a_{t},\ r_{t + 1},\ s_{t + 1})(st​, at​, rt+1​, st+1​) 进行自举并在线更新：

v(st)←v(st)+α[rt+1+γv(st+1)−v(st)]v(s_{t}) \leftarrow v(s_{t}) + \alpha \Big[ r_{t + 1} + \gamma v(s_{t + 1}) - v(s_{t}) \Big] v(st​)←v(st​)+α[rt+1​+γv(st+1​)−v(st​)]

式中的 δt(1)=rt+1+γv(st+1)−v(st)\delta_{t}^{(1)} = r_{t + 1} + \gamma v(s_{t + 1}) - v(s_{t})δt(1)​=rt+1​+γv(st+1​)−v(st​) 为 TD 误差项，Gt(1)=Rt+1+γv(St+1)G_{t}^{(1)} = R_{t + 1} + \gamma v(S_{t + 1})Gt(1)​=Rt+1​+γv(St+1​) 被称为 TD 目标，由于：

v(s)=E(Gt∣St=s)=E(Rt+1+γGt+1∣St=s)=E(Rt+1+γv(St+1)∣St=s)v(s) = \mathcal{E}(G_{t} \mid S_{t} = s) = \mathcal{E}(R_{t + 1} + \gamma G_{t + 1} \mid S_{t} = s) = \mathcal{E}(R_{t + 1} + \gamma v(S_{t + 1}) \mid S_{t} = s) v(s)=E(Gt​∣St​=s)=E(Rt+1​+γGt+1​∣St​=s)=E(Rt+1​+γv(St+1​)∣St​=s)

因此可以使用 Gt(1)G_{t}^{(1)}Gt(1)​ 的采样值 rt+1+γv(st+1)r_{t + 1} + \gamma v(s_{t + 1})rt+1​+γv(st+1​) 来代替 GtG_{t}Gt​ 的采样值 gtg_{t}gt​，以上推导基于：

(Y⊥Z∣X)⇒Ex[Ey(Y=y∣X=x)∣Z=z]=∫Xp(x∣z)dx∫Yyp(y∣x)dx=∫Yydy∫Xp(y∣x)p(x∣z)dx=∫Yydy∫Xp(y∣x, z)p(x∣z)dx=∫Yydy∫Xp(x, y∣z)dx=∫Yyp(y∣z)dy=Ey(Y=y∣Z=z)\begin{aligned} (Y \perp Z \mid X) &\Rightarrow \mathcal{E}_{x} \Big[ \mathcal{E}_{y} (Y = y \mid X = x) \mid Z = z \Big] = \int_{\mathcal{X}} p(x \mid z) dx \int_{\mathcal{Y}} y p(y \mid x) dx \\[5mm] &= \int_{\mathcal{Y}} y dy \int_{\mathcal{X}} p(y \mid x) p(x \mid z) dx = \int_{\mathcal{Y}} y dy \int_{\mathcal{X}} p(y \mid x,\ \xcancel{z}) p(x \mid z) dx \\[5mm] &= \int_{\mathcal{Y}} y dy \int_{\mathcal{X}} p(x,\ y \mid z) dx = \int_{\mathcal{Y}} y p(y \mid z) dy = \mathcal{E}_{y}(Y = y \mid Z = z) \end{aligned} (Y⊥Z∣X)​⇒Ex​[Ey​(Y=y∣X=x)∣Z=z]=∫X​p(x∣z)dx∫Y​yp(y∣x)dx=∫Y​ydy∫X​p(y∣x)p(x∣z)dx=∫Y​ydy∫X​p(y∣x, z​)p(x∣z)dx=∫Y​ydy∫X​p(x, y∣z)dx=∫Y​yp(y∣z)dy=Ey​(Y=y∣Z=z)​

但注意只有在 v=vπv = v_{\pi}v=vπ​ 时上述估计才是无偏的，在未收敛到真实 vπv_{\pi}vπ​ 前这种自举估计是有偏的，但是相对于蒙特卡洛方法降低了估计的方差，因此总体上降低了估计的误差。

多步时序差分

以上方法使用单步的 TD 目标 Gt(1)G_{t}^{(1)}Gt(1)​，类似地，通过多步信息可以得到多步的 TD 目标 G(k)G^{(k)}G(k)

Gt(k)=Rt+1+γRt+2+⋯+γkv(St+k) ⇒ v(s)=E(Gt(k)∣St=s)G_{t}^{(k)} = R_{t + 1} + \gamma R_{t + 2} + \cdots + \gamma^{k} v(S_{t + k})\ \Rightarrow\ v(s) = \mathcal{E}(G_{t}^{(k)} \mid S_{t} = s) Gt(k)​=Rt+1​+γRt+2​+⋯+γkv(St+k​) ⇒ v(s)=E(Gt(k)​∣St​=s)

以及多步 TD 误差 δt(k)=rt+1+γrt+2+⋯+γkv(st+1)−v(st)\delta_{t}^{(k)} = r_{t + 1} + \gamma r_{t + 2} + \cdots + \gamma^{k} v(s_{t + 1}) - v(s_{t})δt(k)​=rt+1​+γrt+2​+⋯+γkv(st+1​)−v(st​)，基于此可以实现多步 TD 方法：

v(st)←v(st)+αδt(k)v(s_{t}) \leftarrow v(s_{t}) + \alpha \delta_{t}^{(k)} v(st​)←v(st​)+αδt(k)​

相较于单步 TD 方法，降低了估计的偏差，而相较于蒙特卡洛方法（T−t\mathrm{T} - tT−t 步 TD）又可以降低估计的方差。
