熵正则方法

为了防止策略网络输出的概率集中在一个动作上，平衡探索性能，可以通过熵来衡量策略分布的不确定性：

Ent⁡(πθ∣s)=−Ea∼πθ(⋅∣s)ln⁡πθ(a∣s)=−∑aπθ(a∣s)ln⁡πθ(a∣s)\operatorname{Ent}(\pi_{\theta} \mid s) = -\mathcal{E}_{a \sim \pi_{\theta}(\cdot \mid s)} \ln \pi_{\theta}(a \mid s) = -\sum_{a} \pi_{\theta}(a \mid s) \ln \pi_{\theta}(a \mid s) Ent(πθ​∣s)=−Ea∼πθ​(⋅∣s)​lnπθ​(a∣s)=−a∑​πθ​(a∣s)lnπθ​(a∣s)

将策略熵作为正则项添加到优化目标中，优化问题变为：

max⁡θJλ(θ)=J(θ)+λΩ(θ)=J(θ)+λEs0Ea0Es1Ea1⋯EsTEaT[∑t=0TγtEnt⁡(πθ∣st)]\max_{\theta} J_{\lambda}(\theta) = J(\theta) + \lambda \Omega(\theta) = J(\theta) + \lambda \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \left[ \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \operatorname{Ent}(\pi_{\theta} \mid s_{t}) \right] θmax​Jλ​(θ)=J(θ)+λΩ(θ)=J(θ)+λEs0​​Ea0​​Es1​​Ea1​​⋯EsT​​EaT​​[t=0∑T​γtEnt(πθ​∣st​)]

在使用策略梯度进行策略提升时，熵正则项的梯度可以计算为：

1},\ a_{t - 1}) \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \nabla_{\theta} \operatorname{Ent}(\pi_{\theta} \mid s_{t}) \right] \\[7mm] &= \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \left[ \left( \sum_{t = 0}^{\mathrm{T}} \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \right) \cdot \left( \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \operatorname{Ent}(\pi_{\theta} \mid s_{t}) \right) + \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \nabla_{\theta} \operatorname{Ent}(\pi_{\theta} \mid s_{t}) \right] \end{aligned} ∇θ​Ω(θ)​=∇θ​s0​∑​a0​∑​s1​∑​a1​∑​⋯sT​∑​aT​∑​[b0​(s0​)t=0∏T​πθ​(at​∣st​)t=1∏T​p(st​∣st−1​, at−1​)t=0∑T​γtEnt(πθ​∣st​)]=s0​∑​a0​∑​s1​∑​a1​∑​⋯sT​∑​aT​∑​[b0​(s0​)(∇θ​t=0∏T​πθ​(at​∣st​))t=1∏T​p(st​∣st−1​, at−1​)t=0∑T​γtEnt(πθ​∣st​)]+s0​∑​a0​∑​s1​∑​a1​∑​⋯sT​∑​aT​∑​[b0​(s0​)t=0∏T​πθ​(at​∣st​)t=1∏T​p(st​∣st−1​, at−1​)t=0∑T​γt∇θ​Ent(πθ​∣st​)]=Es0​​Ea0​​Es1​​Ea1​​⋯EsT​​EaT​​[(t=0∑T​∇θ​lnπθ​(at​∣st​))⋅(t=0∑T​γtEnt(πθ​∣st​))+t=0∑T​γt∇θ​Ent(πθ​∣st​)]​

其中策略熵的梯度为：

∇θEnt⁡(πθ∣s)=−∑a[ln⁡πθ(a∣s)∇θπθ(a∣s)+πθ(a∣s)∇θln⁡πθ(a∣s)]=−∑a[ln⁡πθ(a∣s)∇θπθ(a∣s)+∇θπθ(a∣s)]=−∑aln⁡πθ(a∣s)∇θπθ(a∣s)=−∑aln⁡πθ(a∣s)πθ(a∣s)∇θln⁡πθ(a∣s)=−Ea∼π(⋅∣s)[ln⁡πθ(a∣s)∇θln⁡πθ(a∣s)]\begin{aligned} \nabla_{\theta} \operatorname{Ent}(\pi_{\theta} \mid s) &= -\sum_{a} \Big[ \ln \pi_{\theta}(a \mid s) \nabla_{\theta} \pi_{\theta}(a \mid s) + \pi_{\theta}(a \mid s) \nabla_{\theta} \ln \pi_{\theta}(a \mid s) \Big] \\[7mm] &= -\sum_{a} \Big[ \ln \pi_{\theta}(a \mid s) \nabla_{\theta} \pi_{\theta}(a \mid s) + \nabla_{\theta} \pi_{\theta}(a \mid s) \Big] \\[7mm] &= -\sum_{a} \ln \pi_{\theta}(a \mid s) \nabla_{\theta} \pi_{\theta}(a \mid s) = -\sum_{a} \ln \pi_{\theta}(a \mid s) \pi_{\theta}(a \mid s) \nabla_{\theta} \ln \pi_{\theta}(a \mid s) \\[7mm] &= -\mathcal{E}_{a \sim \pi(\cdot \mid s)} \Big[ \ln \pi_{\theta}(a \mid s) \nabla_{\theta} \ln \pi_{\theta}(a \mid s) \Big] \end{aligned} ∇θ​Ent(πθ​∣s)​=−a∑​[lnπθ​(a∣s)∇θ​πθ​(a∣s)+πθ​(a∣s)∇θ​lnπθ​(a∣s)]=−a∑​[lnπθ​(a∣s)∇θ​πθ​(a∣s)+∇θ​πθ​(a∣s)]=−a∑​lnπθ​(a∣s)∇θ​πθ​(a∣s)=−a∑​lnπθ​(a∣s)πθ​(a∣s)∇θ​lnπθ​(a∣s)=−Ea∼π(⋅∣s)​[lnπθ​(a∣s)∇θ​lnπθ​(a∣s)]​

基于策略熵的梯度可以得到熵正则项的后一项的梯度：

Es0Ea0Es1Ea1⋯EsTEaT[∑t=0Tγt∇θEnt⁡(πθ∣st)]=−Es0Ea0Es1Ea1⋯EsTEaT[∑t=0TγtEat∼π(⋅∣st)[ln⁡πθ(at∣st)∇θln⁡πθ(at∣st)]]=−∑t=0TγtEs0Ea0Es1Ea1⋯EstEat[ln⁡πθ(at∣st)∇θln⁡πθ(at∣st)]\begin{aligned} &\mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \left[ \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \nabla_{\theta} \operatorname{Ent} (\pi_{\theta} \mid s_{t}) \right] \\[7mm] = &-\mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \left[ \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \mathcal{E}_{a_{t} \sim \pi(\cdot \mid s_{t})} \Big[ \ln \pi_{\theta}(a_{t} \mid s_{t}) \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \Big] \right] \\[7mm] = &-\sum_{t = 0}^{\mathrm{T}} \gamma^{t} \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{t}} \mathcal{E}_{a_{t}} \Big[ \ln \pi_{\theta}(a_{t} \mid s_{t}) \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \Big] \end{aligned} ==​Es0​​Ea0​​Es1​​Ea1​​⋯EsT​​EaT​​[t=0∑T​γt∇θ​Ent(πθ​∣st​)]−Es0​​Ea0​​Es1​​Ea1​​⋯EsT​​EaT​​[t=0∑T​γtEat​∼π(⋅∣st​)​[lnπθ​(at​∣st​)∇θ​lnπθ​(at​∣st​)]]−t=0∑T​γtEs0​​Ea0​​Es1​​Ea1​​⋯Est​​Eat​​[lnπθ​(at​∣st​)∇θ​lnπθ​(at​∣st​)]​

通过和策略梯度中类似的方法，乘积因子 ∇θln⁡πθ(at∣st)Ent⁡(πθ∣sτ)\nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \operatorname{Ent}(\pi_{\theta} \mid s_{\tau})∇θ​lnπθ​(at​∣st​)Ent(πθ​∣sτ​) 在 t>τt > \taut>τ 时的期望：

Es0Ea0Es1Ea1⋯EsTEaT[∇θln⁡πθ(at∣st)Ent⁡(πθ∣sτ)]=Es0Ea0Es1Ea1⋯Est[Ent⁡(πθ∣sτ)Eat∇θln⁡πθ(at∣st)⏟0]=0\mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \Big[ \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \operatorname{Ent}(\pi_{\theta} \mid s_{\tau}) \Big] = \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{t}} \Big[ \operatorname{Ent}(\pi_{\theta} \mid s_{\tau}) \underset{0}{\underbrace{\mathcal{E}_{a_{t}} \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t})}} \Big] = 0 Es0​​Ea0​​Es1​​Ea1​​⋯EsT​​EaT​​[∇θ​lnπθ​(at​∣st​)Ent(πθ​∣sτ​)]=Es0​​Ea0​​Es1​​Ea1​​⋯Est​​[Ent(πθ​∣sτ​)0Eat​​∇θ​lnπθ​(at​∣st​)​​]=0

基于此可以将熵正则项的前一项梯度简化为：

Es0Ea0Es1Ea1⋯EstEat[∑t=0T∇θln⁡πθ(at∣st)⋅γtEst+1Eat+1⋯EsTEaT∑τ=tTγτ−tEnt⁡(πθ∣sτ)⏟ξπθ(t)(st, at)]=∑t=0TγtEs0Ea0Es1Ea1⋯EstEat[∇θln⁡πθ(at∣st)ξπθ(t)(st, at)]\begin{aligned} &\mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{t}} \mathcal{E}_{a_{t}} \bigg[ \sum_{t = 0}^{\mathrm{T}} \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \cdot \gamma^{t} \underset{\xi_{\pi_{\theta}}^{(t)}(s_{t},\ a_{t})}{\underbrace{\mathcal{E}_{s_{t + 1}} \mathcal{E}_{a_{t + 1}} \cdots \mathcal{E}_{s_{\mathrm{T}}} \mathcal{E}_{a_{\mathrm{T}}} \sum_{\tau = t}^{\mathrm{T}} \gamma^{\tau - t} \operatorname{Ent}(\pi_{\theta} \mid s_{\tau})}} \bigg] \\[10mm] = & \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{t}} \mathcal{E}_{a_{t}} \Big[ \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \xi_{\pi_{\theta}}^{(t)}(s_{t},\ a_{t}) \Big] \end{aligned} =​Es0​​Ea0​​Es1​​Ea1​​⋯Est​​Eat​​[t=0∑T​∇θ​lnπθ​(at​∣st​)⋅γtξπθ​(t)​(st​, at​)Est+1​​Eat+1​​⋯EsT​​EaT​​τ=t∑T​γτ−tEnt(πθ​∣sτ​)​​]t=0∑T​γtEs0​​Ea0​​Es1​​Ea1​​⋯Est​​Eat​​[∇θ​lnπθ​(at​∣st​)ξπθ​(t)​(st​, at​)]​

最终得到完整的熵正则项的梯度：

∇θΩ(θ)=∑t=0TγtEs0Ea0Es1Ea1⋯EstEat[∇θln⁡πθ(at∣st)(ξπθ(t)(st, at)−ln⁡πθ(at∣st))]\nabla_{\theta} \Omega(\theta) = \sum_{t = 0}^{\mathrm{T}} \gamma^{t} \mathcal{E}_{s_{0}} \mathcal{E}_{a_{0}} \mathcal{E}_{s_{1}} \mathcal{E}_{a_{1}} \cdots \mathcal{E}_{s_{t}} \mathcal{E}_{a_{t}} \bigg[ \nabla_{\theta} \ln \pi_{\theta}(a_{t} \mid s_{t}) \Big( \xi_{\pi_{\theta}}^{(t)}(s_{t},\ a_{t}) - \ln \pi_{\theta}(a_{t} \mid s_{t}) \Big) \bigg] ∇θ​Ω(θ)=t=0∑T​γtEs0​​Ea0​​Es1​​Ea1​​⋯Est​​Eat​​[∇θ​lnπθ​(at​∣st​)(ξπθ​(t)​(st​, at​)−lnπθ​(at​∣st​))]
