General Structure

MAGIC designs a Scheduler and a Message Processor to determine when, whom and what message to communicate

which take the MLP-encoded local observation and LSTM-merged historical information as input

hit, cit=LSTM⁡(e(oit), hit−1, cit−1)⟹mit(0)=hith_{i}^{t},\ c_{i}^{t} = \operatorname{LSTM}(e(o_{i}^{t}),\ h_{i}^{t - 1},\ c_{i}^{t - 1}) \quad \Longrightarrow \quad m_{i}^{t(0)} = h_{i}^{t} hit​, cit​=LSTM(e(oit​), hit−1​, cit−1​)⟹mit(0)​=hit​

The integrated message is further fed to the policy head and value head, achieving better coordination in large-scale MAS

Communication

MAGIC consists of multiple stages of message scheduling and message processing, namely multi-round communication

Scheduler

The Scheduler first adopts a GAT encoder to encode local or global information for an agent at each time step

eit=ELU⁡[∑j∈Nti∪iαijsWsmjt(0)]αijs∝exp⁡{LReLU⁡[as⊤Wsmit(0)+bs⊤Wsmjt(0)]}e_{i}^{t} = \operatorname{ELU} \left[ \sum_{j \in N_{t}^{i} \cup i} \alpha_{ij}^{s} W_{s} m_{j}^{t(0)} \right] \quad \alpha_{ij}^{s} \propto \exp \left\{ \operatorname{LReLU} \left[ a_{s}^{\top} W_{s} m_{i}^{t(0)} + b_{s}^{\top} W_{s} m_{j}^{t(0)} \right] \right\} eit​=ELU⎣⎢⎡​j∈Nti​∪i∑​αijs​Ws​mjt(0)​⎦⎥⎤​αijs​∝exp{LReLU[as⊤​Ws​mit(0)​+bs⊤​Ws​mjt(0)​]}

where the NtiN_{t}^{i}Nti​ is the predefined neighboring agents of iii and the attention weight αijs\alpha_{ij}^{s}αijs​ is only calculated on j∈Ntij \in N_{t}^{i}j∈Nti​

The communication scheduling graph at each stage is derived from pairwise node features and gumbel softmax sampling

gijt(l)=gum⁡[MLP⁡s(l)(eit ∥ ejt)]∈{0, 1}g_{ij}^{t(l)} = \operatorname{gum} \left[ \operatorname{MLP}_{s}^{(l)} \Big( e_{i}^{t}\ \|\ e_{j}^{t} \Big) \right] \in \{ 0,\ 1 \} gijt(l)​=gum[MLPs(l)​(eit​ ∥ ejt​)]∈{0, 1}

Processor

The Message Processor performs message passing and integration based on the communication scheduling graph

mit(l)=ELU⁡[∑j∈Ntiαijp(l)Wp(l)mjt(l−1)]αijp∝exp⁡{gijt(l)LReLU⁡[ap(l)⊤Wp(l)mit(l−1)+bp(l)⊤Wp(l)mjt(l−1)]}m_{i}^{t(l)} = \operatorname{ELU} \left[ \sum_{j \in N_{t}^{i}} \alpha_{ij}^{p(l)} W_{p}^{(l)} m_{j}^{t(l - 1)} \right] \quad \alpha_{ij}^{p} \propto \exp \left\{ g_{ij}^{t(l)} \operatorname{LReLU} \left[ a_{p}^{(l) \top} W_{p}^{(l)} m_{i}^{t(l - 1)} + b_{p}^{(l) \top} W_{p}^{(l)} m_{j}^{t(l - 1)} \right] \right\} mit(l)​=ELU⎣⎢⎡​j∈Nti​∑​αijp(l)​Wp(l)​mjt(l−1)​⎦⎥⎤​αijp​∝exp{gijt(l)​LReLU[ap(l)⊤​Wp(l)​mit(l−1)​+bp(l)⊤​Wp(l)​mjt(l−1)​]}
